# ProyectoFinalG3
Proyecto para ISWZ3208 - Calidad de Software.
## Estructura
- `src/main/java`: Código fuente.
- `src/test/java`: Pruebas.
- `docs/`: Documentación (Plan de Acción, Informe Final, reportes).
## Instrucciones
- Ejecutar: `mvn clean install`
- Generar informes: `mvn checkstyle:check spotbugs:check jacoco:report`
